# Beyond-Nails-Spa

**Live site:** https://beyond-nails-spa.vercel.app

Next.js site lives in [`nextjs-app`](nextjs-app). Business info, service menu, reviews, and gallery content are all in [`nextjs-app/lib/data.ts`](nextjs-app/lib/data.ts). To swap in real photos, drop them in `public/gallery/` and set the `image` field on items in `galleryItems`.
